## 클로디를 소개합니다 🍀

클로디는 감사일기를 작성하고 AI 로디가 답장을 해주는 서비스 입니다! 감사하는 마음을 통해 우리는 삶에서 더 많은 긍정적인 순간들을 발견하고,
이를 통해 더 행복하고 행운이 따르는 삶을 살 수 있게 됩니다. 클로디가 행운의 친구가 되어 감사한 마음을 응원해드릴게요!
